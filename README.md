# Project 2: Exploratory Data Analysis (EDA) on Order Dataset

## Overview
This project performs Exploratory Data Analysis (EDA) on an order dataset to uncover trends, patterns, and business insights.

## Objectives
- Understand dataset structure and quality
- Generate descriptive statistics
- Analyze sales and order behavior
- Explore time-based trends
- Identify key business insights through visualization

## Dataset Features
- Date
- Quantity
- Unit Price
- Items In Cart
- Total Price
- Year, Month, Day, Week Number

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data inspection and validation
- Summary statistics
- Trend analysis over time
- Distribution analysis
- Correlation and relationship exploration
- Business insight generation

## Deliverables
- EDA Notebook
- Visualizations
- PDF Report
- Key Findings and Recommendations

## Learning Outcomes
This project demonstrates how exploratory data analysis helps transform raw transactional data into actionable business insights.
